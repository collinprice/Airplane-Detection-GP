Airplane Detector using ECJ
By Collin Price

Description
This application uses the Java-based Evolutionary Computation Research System (ECJ) to train a spacial image analyzer to locate airplanes. The ECJ project can be found at http://cs.gmu.edu/~eclab/projects/ecj/.

The details of this project are outlined in Airplane-Detection-using-GP.docx.
